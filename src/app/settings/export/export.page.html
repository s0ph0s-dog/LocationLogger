Export Data

Your data will be downloaded as a CSV file. Each row is one log entry. The columns are:

  1. datetime: RFC 3339 timestamp with your local offset at time of logging (not UTC)
  2. latitude: Latitude coordinate (decimal)
  3. longitude: Longitude coordinate (decimal)
  4. country: The country name
  5. level1: The state or province name (in technical terms, "administrative division level 1" for each country)
